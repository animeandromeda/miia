# Miia

Lightweight. Fast. Scalable. Miia stub.  

Only 90 LOC, based on tinyhttp ensures high performance and concurrency.  
Small footprint, only 2 dependecies.

## Setup

```sh
pnpm i
```

## Run

```sh
node app.js
```
